```
# 🤖💰 FinanBot - Assistente IA para Finanças Pessoais

**Revolucione sua gestão financeira conversando com seu dinheiro através de Inteligência Artificial**

## 🚀 O que é o FinanBot?

FinanBot é uma plataforma SaaS completa que permite conversar com suas finanças em linguagem natural. Integrado ao Open Finance brasileiro, conecta-se automaticamente aos seus bancos e oferece análises inteligentes através de IA.

### ✨ Principais Funcionalidades

- 🤖 **Chatbot Inteligente**: Pergunte sobre gastos, receitas e planejamento financeiro
- 🏦 **Open Finance**: Conexão automática com bancos brasileiros (Itaú, Nubank, Bradesco, etc.)
- 📊 **Análises com IA**: Insights personalizados powered by OpenAI GPT-4
- 🔌 **APIs REST**: Endpoints para integração com outras ferramentas
- 🛠️ **MCP Server**: Integração nativa com Claude, Cursor e assistentes de IA
- ⚡ **Real-time**: Chat em tempo real com WebSockets
- 🔒 **Seguro**: Conformidade com regulamentações do Open Finance

### 🛠️ Stack Tecnológica

**Frontend:**
- Next.js 14 com App Router
- TypeScript + Tailwind CSS
- Chart.js para visualizações
- Framer Motion para animações
- Socket.io para tempo real

**Backend:**
- Node.js + Express.js
- TypeScript + Prisma ORM
- PostgreSQL + Redis
- JWT Authentication
- OpenAI API integration
- Socket.io server

**DevOps:**
- Docker + Docker Compose
- Nginx reverse proxy
- SSL/TLS configurado
- Scripts de deploy automatizado

**Integrações:**
- Open Finance API
- OpenAI GPT-4
- MCP (Multi-Cloud Platform) Server
- Suporte para Claude, Cursor, ChatGPT

### 🏦 Bancos Suportados

Itaú, Nubank, Bradesco, Banco do Brasil, Santander, Caixa, XP Banking, BTG Pactual e mais.

### 🎯 Use Cases

- Análise de gastos por categoria
- Planejamento orçamentário inteligente
- Alertas de gastos excessivos
- Previsões financeiras
- Integração com ferramentas de produtividade
- Automação financeira via APIs

### 🚀 Deploy

Pronto para produção com Docker Compose. Inclui configuração completa de:
- PostgreSQL + Redis
- Nginx com SSL
- Backend + Frontend
- Variáveis de ambiente seguras
- Scripts de setup automatizado
```
